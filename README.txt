SPXL Entry Calculator - Streamlit Webapp

Wie starten:
1. Stelle sicher, dass Python installiert ist.
2. Installiere Anforderungen: pip install -r requirements.txt
3. Starte die App: streamlit run app.py
4. Öffne im Browser: http://localhost:8501

Dateien:
- app.py : Streamlit-App
- requirements.txt : benötigte Pakete
- spxl_entry_points.csv : wird generiert beim Download innerhalb der App
